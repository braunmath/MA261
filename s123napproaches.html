    Based on the explorations above, we might conjecture that the sum \(1+2+3+\cdots+n\) is given by a quadratic function of \(n\text{.}\) So, suppose that there are constants \(a\text{,}\) \(b\text{,}\) and \(c\) such that the equality
    \begin{equation*} 1+2+3+\cdots+n = an^2 + bn + c \end{equation*}
    holds for all positive integers \(n\text{.}\) Using the small values we computed earlier for \(n=1,2,3\text{,}\) we can set up a system of equations to solve for \(a\text{,}\) \(b\text{,}\) and \(c\text{.}\)
    \begin{align*} 1 \amp = a(1)^2 + b(1) + c\\ 1+2 \amp = a(2)^2 + b(2) + c\\ 1+2+3 \amp = a(3)^2 + b(3) + c \end{align*}
    Solving for \(a,b\text{,}\) and \(c\) (for example, by back substitution), we find that \(a=\frac{1}{2}, b=\frac{1}{2}, c=0\text{,}\) so we have the following conjecture.

    Conjecture 2.2.7.

    \begin{equation*} 1+2+3+\cdots+n = \frac{1}{2}n^2 + \frac{1}{2}n = \frac{n(n+1)}{2} \end{equation*}
