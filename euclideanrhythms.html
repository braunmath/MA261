    We will now introduce an algorithm similar to the Euclidean algorithm that produces rhythms. This algorithm will be called \(\textrm{ER}\) for Euclidean Rhythm, it will have input a pair of integers \(k,n\) and it will output a distance sequence defining a rhythm.
    ðŸ”—

    Algorithm 4.6.4. Euclidean Rhythm Algorithm.

    Suppose \(n\gt k\) are positive integers. The algorithm \(\textrm{ER}(k,n)\) runs as follows.
    • If \(k\) evenly divides \(n\text{,}\) then return
      \begin{equation*} (\underbrace{\frac{n}{k},\frac{n}{k},\ldots, \frac{n}{k}}_{k\text{ times}}) \, . \end{equation*}

      ðŸ”—
    • Else, write \(n=kq+r\) with \(0\leq r\lt k\) and compute the sequence
      \begin{equation*} (x_1,\ldots,x_r)= \textrm{ER}(r,k) \, . \end{equation*}
      Then, return the sequence
      \begin{equation*} (\underbrace{\left\lfloor \frac{n}{k} \right\rfloor,\ldots, \left\lfloor \frac{n}{k} \right\rfloor}_{x_1-1\text{ times}}, \left\lceil \frac{n}{k} \right\rceil, \underbrace{\left\lfloor \frac{n}{k} \right\rfloor,\ldots, \left\lfloor \frac{n}{k} \right\rfloor}_{x_2-1\text{ times}}, \left\lceil \frac{n}{k} \right\rceil,\ldots,\underbrace{\left\lfloor \frac{n}{k} \right\rfloor,\ldots, \left\lfloor \frac{n}{k} \right\rfloor}_{x_r-1\text{ times}}, \left\lceil \frac{n}{k} \right\rceil) \, . \end{equation*}

    Theorem 4.6.7.

    The output of \(\textrm{ER}(k,n)\) is a sequence of \(k\) integers that sum to \(n\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    We will use strong induction on \(k\text{.}\) The base case is when \(k\mid n\text{,}\) for which we have
    \begin{equation*} \textrm{ER}(k,n)=(\underbrace{\frac{n}{k},\frac{n}{k},\ldots, \frac{n}{k}}_{k\text{ times}}) \end{equation*}
    and the result clearly holds in this case.
    ðŸ”—
    For the induction step, suppose \(k\nmid n\) and write \(n=kq+r\) with \(0\leq r\lt k\text{.}\) Since \(r\lt k\text{,}\) by induction we have that if
    \begin{equation*} (x_1,\ldots,x_r)=\textrm{ER}(r,k) \end{equation*}
    then \(x_1+\cdots +x_r=k\text{.}\) Therefore, the sequence
    \begin{equation*} \textrm{ER}(k,n)=(\underbrace{\left\lfloor \frac{n}{k} \right\rfloor,\ldots, \left\lfloor \frac{n}{k} \right\rfloor}_{x_1-1\text{ times}}, \left\lceil \frac{n}{k} \right\rceil, \underbrace{\left\lfloor \frac{n}{k} \right\rfloor,\ldots, \left\lfloor \frac{n}{k} \right\rfloor}_{x_2-1\text{ times}}, \left\lceil \frac{n}{k} \right\rceil,\ldots,\underbrace{\left\lfloor \frac{n}{k} \right\rfloor,\ldots, \left\lfloor \frac{n}{k} \right\rfloor}_{x_r-1\text{ times}}, \left\lceil \frac{n}{k} \right\rceil) \end{equation*}
    has \(k\) entries and has a sum given by
    \begin{align*} r\left\lceil \frac{n}{k} \right\rceil + \left\lfloor \frac{n}{k} \right\rfloor \sum_{i=1}^r (x_i-1) \amp = r\left\lceil \frac{n}{k} \right\rceil + \left\lfloor \frac{n}{k} \right\rfloor (k-r)\\ \amp = r\left( 1+\left\lfloor \frac{n}{k}\right\rfloor \right) + \left\lfloor \frac{n}{k}\right\rfloor (k-r)\\ \amp = r+ k\left\lfloor \frac{n}{k}\right\rfloor \\ \amp = r+kq \\ \amp = n \end{align*}
    which completes the proof.
