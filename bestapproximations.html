    Section 6.2 Best Approximations

    We gave an embodied argument in the previous section justifying that we can approximate any real number by rational numbers with denominator given by a powers of \(10\text{.}\) This is the method of decimal expansions, and it is a very natural method for approximation of real numbers by rationals. A natural question to ask is the following: If we are allowed to use any rational number with denominator between \(1\) and \(q\text{,}\) what is the best approximation to a given real number \(s\) that we can get? There are two different ways to interpret the meaning of "best" in this case, and these give rise to two different definitions of best approximations.
    🔗

    Definition 6.2.1.

    Given a real number \(s\text{,}\) we say that a rational number \(p/q\) is a best approximation of the first kind to \(s\) if for any other rational number \(r/t\) with denominator \(t\leq q\text{,}\) we have
    \begin{equation*} \left|s-\frac{p}{q}\right|\lt \left|s-\frac{r}{t}\right| \, . \end{equation*}
    We say that a rational number \(p/q\) is a best approximation of the second kind to \(s\) if for any other rational number \(r/t\) with denominator \(t\leq q\text{,}\) we have
    \begin{equation*} \left|q s-p\right|\lt \left|t s-r\right| \, . \end{equation*}
    🔗
    🔗
    For an approximation of the first kind, suppose that we have
    \begin{equation*} \left|s-\frac{p}{q}\right|\approx 0 \, . \end{equation*}
    Then we have that
    \begin{equation*} s \approx \frac{p}{q} \, , \end{equation*}
    from which it would follow that
    \begin{equation*} qs \approx p\, , \end{equation*}
    from which it would follow that
    \begin{equation*} \left|qs-p\right|\approx 0 \, , \end{equation*}
    giving an approximation of the second kind. Thus, these two types of best approximations are related, though what they measure is not the same (as we will see).
    🔗

    Consider the following SageMath code that computes the best approximation of the first kind to a real number \(s\text{.}\) Running this in https://sagecell.sagemath.org/ produces best approximations to \(\pi-3\text{,}\) \(\frac{1+\sqrt{5}}{2}-1\text{,}\) and \(\log_2(3/2)\text{.}\)
    🔗
    def firstbestapproximation(s,q):
    """
    input: a real number s between 0 and 1 and a maximum denominator q
    output: the rational number with denominator at most q that is the best approximation to s of the first kind
    """
    best = 0
    for j in srange(1,q+1):
        for k in srange(0,j+1):
            if abs(s-k/j) <= abs(s-best):
                best = k/j
    return best

print(firstbestapproximation(pi-3,10))

print(firstbestapproximation((1+sqrt(5))/2-1,50))

print(firstbestapproximation(log(3/2,2),60))

    The following code computes the best approximation of the second kind to a real number \(s\text{.}\)
    🔗
    def secondbestapproximation(s,q):
    """
    input: a real number s between 0 and 1 and a maximum denominator q
    output: the rational number with denominator at most q that is the best approximation to s of the second kind
    """ 
    num_best = 0
    denom_best = 1
    for j in srange(1,q+1):
        for k in srange(0,j+1):
            if abs(j*s-k) <= abs(denom_best*s-num_best):
                num_best = k
                denom_best = j
    return num_best/denom_best

print(secondbestapproximation(pi-3,10))

print(secondbestapproximation((1+sqrt(5))/2-1,50))

print(secondbestapproximation(log(3/2,2),60))

    The following code demonstrates that these two types of approximations are not always the same. Specifically, the first and second best approximations to \(\pi-3\) with denominator at most \(100\) are different.
    🔗
    def firstbestapproximation(s,q):
    """
    input: a real number s between 0 and 1 and a maximum denominator q
    output: the rational number with denominator at most q that is the best approximation to s of the first kind
    """
    best = 0
    for j in srange(1,q+1):
        for k in srange(0,j+1):
            if abs(s-k/j) <= abs(s-best):
                best = k/j
    return best

def secondbestapproximation(s,q):
    """
    input: a real number s between 0 and 1 and a maximum denominator q
    output: the rational number with denominator at most q that is the best approximation to s of the second kind
    """ 
    numbest = 0
    denombest = 1
    for j in srange(1,q+1):
        for k in srange(0,j+1):
            if abs(j*s-k) <= abs(denombest*s-numbest):
                numbest = k
                denombest = j
    return numbest/denombest

print(firstbestapproximation(pi-3,100))

print(secondbestapproximation(pi-3,100))
    A natural question to ask is how these two types of best approximations are related. The answer is given by the following theorem.
    🔗

    Theorem 6.2.5.

    If \(p/q\) is a best approximation of the second kind to \(s\text{,}\) then it is also a best approximation of the first kind to \(s\text{.}\)
    🔗
    🔗
    We will prove this by contrapositive: this means that we will prove \(P\implies Q\) by showing that \(\neg Q \implies \neg P\) where \(\neg P\) means "not \(P\)", or the negation of \(P\text{.}\) Observe that if we prove that \(p/q\) not being a first kind best approximation implies it is also not a second kind best approximation, then it is impossible for \(p/q\) to be simultaneously a second kind best approximation and not a first kind best approximation.
    🔗

    Checkpoint 6.2.6.

    Discuss the strategy of proof by contrapositive. Does it make sense what we are about to prove, and why this proof strategy works?
    🔗
    🔗

    Proof.

    Suppose that \(p/q\) is not a best approximation of the first kind to \(s\text{.}\) Then there exists \(1\leq q'\leq q\) and \(p'\) such that
    \begin{equation*} \left| s-p'/q'\right| \leq \left| s-p/q\right|\, . \end{equation*}
    Since \(q'\leq q\text{,}\) we can multiply the left-hand side of the displayed equation by \(q'\) and the right-hand side by \(q\) without changing the inequality. Thus, we have
    \begin{equation*} \left| q's-p'\right| \leq \left| q s-p\right|\, , \end{equation*}
    from which it follows that \(p/q\) is not a best approximation of the second kind to \(s\text{.}\)
    🔗
    🔗

    Checkpoint 6.2.7.

    Discuss this proof with your group. Does it make sense? Why or why not?

    Thus, being a best approximation of the second kind is a stronger condition than being a best approximation of the first kind. How do we find these approximations without carrying out a brute force search like we did in the SageMath code above? The answer is that best approximations of the second kind are produced by convergents of continued fractions for \(s\). Thus, we will study continued fractions and their convergents.
