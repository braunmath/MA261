    Section 5.4 Solving Modular Linear Equations

    Now that we have developed a new number system to work with, we can consider standard problems in this new context. For example, the first functions you learn about in school are linear functions, which are functions of the form \(y=mx+b\text{.}\) You learned that if \(m\neq 0\text{,}\) then the function is invertible, and you can solve for \(x\) in terms of \(y\text{.}\) In the context of modular arithmetic, we can ask a similar question: given a linear function \(f(x)=mx+b\text{,}\) can we solve for \(x\) in terms of \(y\) when we are working modulo some integer \(n\text{?}\) For this problem, we will consider two solutions \(x\) to be the same if they are congruent modulo \(n\text{.}\) In other words, we will seek solutions to \([m]x+[b]=y\) where \(x,y\in\mathbb{Z}/n\mathbb{Z}\text{.}\)
    🔗
    As an example, consider the function \(f(x)=3x\) working with the number system \(\mathbb{Z}/5\mathbb{Z}\text{.}\) We can ask the question: given a value of \(y\text{,}\) can we find a value of \(x\) such that \(f(x)=y\text{?}\) For example, what are the solutions to the following equation?
    \begin{equation*} 3x\equiv 0 \bmod 5 \end{equation*}
    We can equivalently write this in \(\mathbb{Z}/5\mathbb{Z}\) as:
    \begin{equation*} [3]x=[0] \end{equation*}
    This is equivalent to finding the values of \(x\) such that \(3x-0=3x\) is divisible by 5. We can check each value of \(x\) in \(\mathbb{Z}/5\mathbb{Z}\) to see which ones work:
    • \(3\cdot 0=0\) is divisible by 5, so \(x=0\) is a solution.
      🔗
    • \(3\cdot 1=3\) is not divisible by 5, so \(x=1\) is not a solution.
      🔗
    • \(3\cdot 2=6\) is not divisible by 5, so \(x=2\) is not a solution.
      🔗
    • \(3\cdot 3=9\) is not divisible by 5, so \(x=3\) is not a solution.
      🔗
    • \(3\cdot 4=12\) is not divisible by 5, so \(x=4\) is not a solution.
      🔗
    Therefore, the only solution to the equation \(3x\equiv 0 \bmod 5\) is \(x=0\text{.}\) What happens if we change the equation to
    \begin{equation*} 3x\equiv 1 \bmod 5\, , \end{equation*}
    which is equivalent over \(\mathbb{Z}/5\mathbb{Z}\) to
    \begin{equation*} [3]x=[1]\, . \end{equation*}
    We can check each value of \(x\) again:
    • \(3\cdot 0=0\) is not congruent to 1 modulo 5, so \(x=0\) is not a solution.
      🔗
    • \(3\cdot 1=3\) is not congruent to 1 modulo 5, so \(x=1\) is not a solution.
      🔗
    • \(3\cdot 2=6\) is congruent to 1 modulo 5, so \(x=2\) is a solution.
      🔗
    • \(3\cdot 3=9\) is not congruent to 1 modulo 5, so \(x=3\) is not a solution.
      🔗
    • \(3\cdot 4=12\) is not congruent to 1 modulo 5, so \(x=4\) is not a solution.
      🔗
    Therefore, the only solution to the equation \(3x\equiv 1 \bmod 5\) is \(x=2\text{.}\) So far, there is a unique solution to each of the equations we have considered, similarly to what we are used to.
    🔗
    However, consider next the equation \(2x \equiv 8 \bmod 10\text{.}\) We can check each value of \(x\) in \(\mathbb{Z}/10\mathbb{Z}\) to see which ones work:
    • \(2\cdot 0=0\) is not congruent to 8 modulo 10, so \(x=0\) is not a solution.
      🔗
    • \(2\cdot 1=2\) is not congruent to 8 modulo 10, so \(x=1\) is not a solution.
      🔗
    • \(2\cdot 2=4\) is not congruent to 8 modulo 10, so \(x=2\) is not a solution.
      🔗
    • \(2\cdot 3=6\) is not congruent to 8 modulo 10, so \(x=3\) is not a solution.
      🔗
    • \(2\cdot 4=8\) is congruent to 8 modulo 10, so \(x=4\) is a solution.
      🔗
    • \(2\cdot 5=10\) is not congruent to 8 modulo 10, so \(x=5\) is not a solution.
      🔗
    • \(2\cdot 6=12\) is not congruent to 8 modulo 10, so \(x=6\) is not a solution.
      🔗
    • \(2\cdot 7=14\) is not congruent to 8 modulo 10, so \(x=7\) is not a solution.
      🔗
    • \(2\cdot 8=16\) is not congruent to 8 modulo 10, so \(x=8\) is not a solution.
      🔗
    • \(2\cdot 9=18\) is congruent to 8 modulo 10, so \(x=9\) is a solution.
      🔗
    Therefore, the solutions to the equation \(2x \equiv 8 \bmod 10\) are \(x=4\) and \(x=9\text{.}\) In this case, there are two solutions to the equation, which is different from what we are used to in the context of real numbers.
    🔗
    As yet another example, consider the equation
    \begin{equation*} 6x\equiv 5 \bmod 8\, , \end{equation*}
    which is equivalent over \(\mathbb{Z}/8\mathbb{Z}\) to
    \begin{equation*} [6]x=[5]\, . \end{equation*}
    In this case, we can immediately see that there are no solutions to this equation, since \(6x-5\) is always an odd number, and therefore it is never divisible by \(8\text{.}\)
    🔗

    So, we have seen that in the context of modular arithmetic, linear equations can have a unique solution, multiple solutions, or no solutions at all. This leads us to the following questions:
    • What are the criteria guaranteeing that a given linear equation has a solution, and when it does, how many solutions does it have?
      🔗
    • Is there a method to find all the solutions to a given linear equation?
      🔗
    Fortunately, we can solve these problems using the Euclidean algorithm and the Bezout identity.
    🔗

    Remark 5.4.2.

    Sometimes when you are reading math, instead of placing the proof of a theorem after the statement of the theorem, an author will lead you through a proof before stating the theorem. We will illustrate this style of writing for the next theorem.
    🔗
    🔗
    Consider the general equation \(ax\equiv b \bmod n\text{.}\) This is equivalent to the equation \([a]x=[b]\) in \(\mathbb{Z}/n\mathbb{Z}\text{,}\) which is equivalent to the equation \(ax-b\equiv 0 \bmod n\text{.}\) So, we need to find the values of \(x\) such that \(ax-b\) is divisible by \(n\text{.}\) This is equivalent to finding the values of \(x\) such that there exists an integer \(k\) such that
    \begin{equation*} ax-b=nk\, , \end{equation*}
    which is equivalent to finding integer solutions \(x\) and \(k\) to the linear equation
    \begin{equation*} ax-nk=ax+n(-k)=b \, . \end{equation*}
    However, we know how to do this! Specifically, we know that the linear equation \(ax+ny=b\) has integer solutions if and only if \(\gcd(a,n)\mid b\text{.}\)
    🔗
    Further, we know that if \(\gcd(a,n)=1\text{,}\) then \([a]\) is a unit in \(\mathbb{Z}/n\mathbb{Z}\text{,}\) so there is a unique solution to the equation \([a]x=[b]\) in this case, namely \(x=[a]^{-1}[b]\text{.}\) If \([a]\) is not a unit and if there is a solution to the equation \([a]x=[b]\text{,}\) then there are multiple solutions. The reason is that if \(x\in\mathbb{Z}/n\mathbb{Z}\) is a solution, then so is \(x+[n/\gcd(a,n)]\text{,}\) since
    \begin{equation*} [a](x+[n/\gcd(a,n)])=[a]x+[a][n/\gcd(a,n)]=[a]x+[0]=[a]x=[b] \, . \end{equation*}
    So, we have shown the following theorem.
    🔗

    Theorem 5.4.3.

    The linear equation \(ax\equiv b \bmod n\text{,}\) or equivalently \([a]x=[b]\) in \(\mathbb{Z}/n\mathbb{Z}\text{,}\) has a solution if and only if \(\gcd(a,n)\mid b\text{.}\) Further, if \(\gcd(a,n)=1\text{,}\) then there is a unique solution to the equation, namely \(x=[a]^{-1}[b]\text{.}\) If \([a]\) is not a unit and if there are solutions to this equation, then there are multiple solutions.
    🔗
