    Observe that we are using set notation above to define these families of numbers. We use "\(:=\)" to mean "is defined to be"; this is helpful when distinguishing between when we define something and when we are indicating equality. When we are defining a set, we often use curly braces \(\{\}\) to indicate the elements of the set and we specify the elements within the braces. One can use special notation within the braces to indicate a pattern for the elements of the set. For example, we can describe the natural numbers as
    \begin{equation*} \mathbb{N} = \{ n \in \mathbb{Z} : n > 0 \} \, . \end{equation*}
    The symbol "\(\in\)" is read as "is an element of" or "belongs to". The ":" is read as "such that", so that the above set is read as "the set of all integers \(n\) such that \(n\) is greater than \(0\)".

    We can combine set builder notation with these definitions to describe sets of numbers with certain divisibility properties. For example, the set of all integers that are divisible by \(5\) but not divisible by \(6\) is written as
    \begin{equation*} \{ n \in \mathbb{Z} : 5 \mid n \text{ and } 6 \nmid n \} \, . \end{equation*}

    \begin{equation*} \{\ldots, -25, -20, -10, -5, 5, 10, 15, 25, \ldots\} \, , \end{equation*}

    Theorem 3.1.12.

    Every natural number greater than 1 is divisible by a prime number.

    Proof.

    This proof closely follows the one given for Proposition 1.6.3 in Cioaba and Lindeâ€™s A Bridge to Advanced Mathematics. We will prove this by strong induction PrincipleÂ 2.6.8. The base case is \(n = 2\text{,}\) and the result is true in this case since \(2\) is prime.
    ðŸ”—
    For the (strong) induction step, let \(m\) be a positive integer and assume that every natural number \(1\leq k\leq m\) has a prime divisor. We want to prove that \(m+1\) also has a prime divisor. If \(m+1\) is prime, then we are done. If not, then \(m+1\) must have some divisor \(d\) such that \(1\lt d\lt m+1\text{,}\) so that means there exists a number \(c\) such that
    \begin{equation*} m+1=cd \, . \end{equation*}
    By our induction hypothesis, \(d\) has a prime divisor. So, there exists a number \(a\) and a prime \(p\) such that
    \begin{equation*} d=ap \, . \end{equation*}
    Therefore, we have that
    \begin{equation*} m+1=cd=cap \end{equation*}
    and thus \(p\) is also a prime divisor of \(m+1\text{,}\) which finishes our proof.
