    Section 5.5 Chinese Remainder Theorem

    Another topic that is important is simultaneously solving systems of linear equations across multiple moduli. In other words, we can consider linear equations with a single variable, but where each linear equation uses a different value for \(n\text{.}\) Let’s look at an example.
    🔗
    Consider the following two equations:
    \begin{align*} x \amp \equiv 1 \bmod 5\\ x \amp \equiv 4 \bmod 7 \end{align*}
    We can ask the question: is there a solution to this system of equations? In this case, we see that \(x=11\) is a solution, since \(11\equiv 1 \bmod 5\) and \(11\equiv 4 \bmod 7\text{.}\) What this means is that the number \(11\) plays two roles simultaneously: we have \([11]=[1]\) in \(\mathbb{Z}/5\mathbb{Z}\) and \([11]=[4]\) in \(\mathbb{Z}/7\mathbb{Z}\text{.}\)
    🔗
    Consider the following more complicated example:
    \begin{align*} x \amp \equiv 12 \bmod 17\\ x \amp \equiv 14 \bmod 23\\ x \amp \equiv 3 \bmod 29\\ x \amp \equiv 97 \bmod 103 \end{align*}
    In this case, we have that \(x=206715\) is a solution. Again, what does this mean? It means that the number \(206715\) plays four roles simultaneously: we have \([206715]=[12]\) in \(\mathbb{Z}/17\mathbb{Z}\text{,}\) \([206715]=[14]\) in \(\mathbb{Z}/23\mathbb{Z}\text{,}\) \([206715]=[3]\) in \(\mathbb{Z}/29\mathbb{Z}\text{,}\) and \([206715]=[97]\) in \(\mathbb{Z}/103\mathbb{Z}\text{.}\)
    🔗
    This leads to the following problem: given \(a_1,\ldots,a_k\) and \(n_1,\ldots,n_k\text{,}\) can we find a number \(x\) such that \([x]=[a_i]\) in \(\mathbb{Z}/n_i\mathbb{Z}\) for each \(i=1,\ldots,k\text{?}\)
    🔗

    Checkpoint 5.5.1.

    Discuss these examples and this question. Do they make sense? Why or why not?
    🔗
    🔗
    The answer to this question is known as the Chinese Remainder Theorem, because problems of this type were solved 1500 years ago by mathematicians in China.
    🔗

    Theorem 5.5.2. Chinese Remainder Theorem.

    Suppose that \(n_1,\ldots,n_k\) are pairwise coprime, i.e., that \(\gcd(n_i,n_j)=1\) for all \(i\neq j\text{.}\) Then for any choice of \(a_i\in \{0,1,\ldots,n_i-1\}\text{,}\) the system of equations given by
    \begin{align*} x \amp \equiv a_1 \bmod n_1\\ x \amp \equiv a_2 \bmod n_2\\ \amp \vdots \\ x \amp \equiv a_k \bmod n_k \end{align*}
    has a unique solution in \(\{0,1,2,\ldots,n_1n_2\cdots n_k-1\}\text{.}\) In other words, there is a unique congruence class \(x\in \mathbb{Z}/n_1n_2\cdots n_k \mathbb{Z}\) whose representatives simultaneously solve these equations.
    🔗
    🔗

    Proof.

    We first prove that if a solution exists, then it is unique. Suppose that \(x\) and \(y\) are both solutions to the above system of equations. This means that for each \(i\text{,}\) we have
    \begin{equation*} x-y \equiv 0 \bmod n_i \end{equation*}
    and thus each \(n_i\) divides \(x-y\text{.}\) Since the \(n_i\) are pairwise coprime, this least common multiple is their product, and this must divide \(x-y\) as well. Thus, \([x]=[y]\) in \(x\in \mathbb{Z}/n_1n_2\cdots n_k \mathbb{Z}\text{.}\)
    🔗
    We next show how to construct a solution to this system of equations. We start with the case of two equations. Bezout’s identity states that we can find \(m_1\) and \(m_2\) such that
    \begin{equation*} n_1m_1+n_2m_2=1 \, . \end{equation*}
    We now claim that we have a solution to the first two equations given by
    \begin{equation*} x = a_1 m_2 n_2 + a_2 m_1 n_1 \, . \end{equation*}
    This is a solution because we can replace \(m_2n_2 = 1-n_1m_1\) in the above equation to get
    \begin{equation*} x \equiv a_1 (1-n_1m_1) + a_2 n_1m_1 \end{equation*}
    and when we reduce this modulo \(n_1\) we get \(a_1\text{.}\) A similar calculation shows that this holds for \(a_2\text{.}\)
    🔗
    When \(k\geq 3\text{,}\) we can use a similar process to the case of two equations. Specifically, let \(N=n_1n_2\cdots n_k\) and let \(S_i=N/n_i\text{.}\) Since the original \(n_i\) are coprime, we have that \(\gcd(S_i,n_i)=1\text{.}\) Thus, there is a multiplicative inverse of \([S_i]\) in \(\mathbb{Z}/n_i\mathbb{Z}\text{,}\) which we will call \([R_i]=[S_i]^{-1}\text{.}\) This multiplicative inverse can be found by solving the Bezout identity, as we have done previously. We define
    \begin{equation*} x = \sum_{i=1}^k a_iR_iS_i \, . \end{equation*}
    We know that
    \begin{equation*} a_iR_iS_i \equiv a_i \bmod n_i \end{equation*}
    for each \(i\) due to the additive inverse relationship. We also have that if \(i\neq j\text{,}\) then
    \begin{equation*} n_i \mid S_j = n_1n_2\cdots n_k/n_j \, . \end{equation*}
    Thus,
    \begin{equation*} x\equiv a_i \bmod n_i \end{equation*}
    for each \(i\text{,}\) as desired.
    🔗
    🔗

    Checkpoint 5.5.3.

    Discuss this proof with your group. Does it make sense? Why or why not?
    🔗
    🔗
    Let’s work through an example to solve such a system of equations. Suppose we have
    \begin{align*} x \amp \equiv 1 \bmod 3\\ x \amp \equiv 2 \bmod 5\\ x \amp \equiv 4 \bmod 7 \end{align*}
    Using the notation from the proof, we have that \(N=3\cdot 5\cdot 7 = 105\text{,}\) \(S_1=35\text{,}\) \(S_2=21\text{,}\) and \(S_3=15\text{.}\) Computing inverses, we have \([35]_3=[2]_3\) (where the 3 subscript indicates the modulus) and thus \(R_1=2\text{.}\) Similarly, \([21]_5=[1]_5\text{,}\) so \(R_2=1\text{.}\) Finally, \([15]_7=[1]_7\text{.}\) Putting this together, we have
    \begin{align*} x \amp = a_1 S_1 R_1 + a_2 S_2 R_2 + a_2 S_2 R_2 \\ \amp = 1\cdot 35\cdot 2 + 2 \cdot 21\cdot 1 + 4 \cdot 15 \cdot 1\\ \amp = 172 \end{align*}
    If we reduce \(172\) modulo \(105\text{,}\) we obtain \(67\text{.}\) Thus, \(67\) is our unique solution in \(\{0,1,2,\ldots, 104\}\text{.}\)
    🔗
