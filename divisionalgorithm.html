    When \(a,b,c\) are integers, the equation
    \begin{equation*} ax+by=c \end{equation*}
    is called a linear Diophantine equation. In general, a Diophantine equation is a polynomial equation with integer coefficients for which we study solutions that are integer values. A linear Diophantine equation is a linear equation with integer coefficients, where we want to find integer solutions. Diophantine equations are named after Diophantus of Alexandria, a Greek mathematician who made significant contributions to number theory.

    \begin{equation*} ax+by=c \end{equation*}
    has integer solutions \(x,y\) if and only if \(c\) is a multiple of the greatest common divisor of \(a\) and \(b\text{.}\) We will prove that this is true later in this chapter, but first we will develop a broad theory regarding both greatest common divisors and the solution of linear Diophantine equations. We will begin with the division algorithm, which is a simple but powerful tool in number theory.

    Theorem 4.1.4. The Division Algorithm.

    Let \(a\) and \(b\) be positive integers, i.e., \(a,b\in \mathbb{N}\text{.}\) Then there exist unique integers \(q\) and \(r\) such that
    \begin{equation*} a = bq + r \end{equation*}
    and \(0 \leq r \lt b\text{.}\)
    ðŸ”—
    ðŸ”—
    Before we prove TheoremÂ 4.1.4, letâ€™s give some names to the quantities involved. We call \(q\) the quotient and \(r\) the remainder when \(a\) is divided by \(b\text{.}\) Observe that \(r=0\) if and only if \(b\) divides \(a\text{,}\) by the definition of divisibility.
    ðŸ”—

    Example 4.1.5.

    Let \(a=17\) and \(b=5\text{.}\) Then by the division algorithm, there exist unique integers \(q\) and \(r\) such that
    \begin{equation*} 17 = 5q + r \end{equation*}
    and \(0 \leq r \lt 5\text{.}\) In this case, we have \(q=3\) and \(r=2\text{,}\) since
    \begin{equation*} 17 = 5\cdot 3 + 2\text{.} \end{equation*}

    While it might seem clear when doing examples that we can always find such integers \(q\) and \(r\) given \(a\) and \(b\text{,}\) neither this nor the uniqueness part of the theorem is not immediately obvious. Letâ€™s provide a proof to verify this formally.
    ðŸ”—

    Proof.

    Consider the set
    \begin{equation*} S=\{j\in \mathbb{N}:jb\gt a\} \, , \end{equation*}
    which is the set of all positive multiples of \(b\) that are greater than \(a\text{.}\) We first want to show that this set is not the empty set. Since \(b\gt 0\text{,}\) we know that \((a+1)b\geq a+1\gt a\text{,}\) so \((a+1)b\in S\text{.}\) Thus,
    \begin{equation*} S\neq \emptyset \, . \end{equation*}
    By the well-ordering principle PrincipleÂ 3.3.7, since \(S\) is a non-empty set of positive integers, it must have a smallest element; call this element \(t\text{.}\)
    ðŸ”—
    Write \(t=q+1\) for some \(q\in \mathbb{Z}_{\geq 0}\text{,}\) and note that \(q \notin S\) by the minimality of \(t\text{.}\) Then we have
    \begin{equation*} (q+1)b = tb \gt a \geq qb \, . \end{equation*}
    Subtracting \(qb\) from each part of this inequality yields
    \begin{equation*} b\gt a - qb \geq 0 \, . \end{equation*}
    So, define \(r = a - qb\text{.}\) Then we have \(0 \leq r \lt b\) and
    \begin{equation*} a = bq + r \, . \end{equation*}
    ðŸ”—
    To see that \(q\) and \(r\) are unique, suppose that there exist integers \(q'\) and \(r'\) such that
    \begin{equation*} a = bq' + r' \, , \end{equation*}
    with \(0 \leq r' \lt b\text{.}\) Then we have
    \begin{equation*} bq + r = bq' + r' \, . \end{equation*}
    Rearranging this equation yields
    \begin{equation*} b(q - q') = r' - r \, . \end{equation*}
    If \(q \neq q'\text{,}\) then \(|q - q'| \geq 1\text{,}\) so \(|b(q - q')| \geq b\text{.}\) But since \(0 \leq r,r' \lt b\text{,}\) we have \(|r' - r| \lt b\text{,}\) which is a contradiction. Thus, we must have \(q = q'\text{,}\) which implies that \(r = r'\) as well. Therefore, the integers \(q\) and \(r\) are unique.
