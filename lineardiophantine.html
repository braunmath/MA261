    Theorem 4.4.1.

    Consider the linear Diophantine equation
    \begin{equation*} ax+by=c \end{equation*}
    for positive integers \(a,b,c\text{.}\) This equation has an integer solution if and only if
    \begin{equation*} \gcd(a,b) \mid c \, . \end{equation*}
    ðŸ”—
    ðŸ”—

    Proof.

    We prove the forward direction first. Suppose
    \begin{equation*} ax+by=c \end{equation*}
    has an integer solution, and let \(d=\gcd(a,b)\text{.}\) Then we have
    \begin{equation*} ax+by=da'x+db'y=d(a'x+b'y)=c \end{equation*}
    and therefore \(d\mid c\text{.}\)
    ðŸ”—
    We next prove the reverse direction. Suppose that \(d=\gcd(a,b)\) and \(dm=c\text{.}\) From the BÃ©zout identity we have a solution
    \begin{equation*} ax+by=d \end{equation*}
    and therefore
    \begin{equation*} a(xm)+b(ym)=dm=c \end{equation*}
    as well. Thus, a solution exists, as claimed.

    Theorem 4.4.4.

    Suppose that \(d=\gcd(a,b)\mid c\) and that \((x_0,y_0)\) is an integer solution to
    \begin{equation*} ax+by=c \, . \end{equation*}
    Let \(b'=b/d\) and \(a'=a/d\text{.}\) Then every integer solution to this equation is of the form
    \begin{equation*} (x,y)=(x_0+b't,y_0-a't)=(x_0,y_0)+t(b',-a') \end{equation*}
    for some \(t\in \Z\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    Suppose that we have two solutions to
    \begin{equation*} ax+by=c\text{,} \end{equation*}
    call them:
    \begin{gather*} ax_1+by_1=c\\ ax_0+by_0=c \end{gather*}
    Taking the difference of these equations we have
    \begin{equation*} a(x_1-x_0)+b(y_1-y_0)=0 \, . \end{equation*}
    Thus, \((x_1-x_0,y_1-y_0)\) is a solution to the linear homogeneous equation
    \begin{equation*} ax+by=0 \end{equation*}
    And we see that any two solutions to \(ax+by=c\) differ by a solution to \(ax+by=0\text{.}\) Thus, every solution is of the form
    \begin{equation*} (x_0,y_0)+\text{ solution to }c=0\text{ case } \, . \end{equation*}
    ðŸ”—
    We can divide by \(d\) to reduce this equation to the form
    \begin{equation*} a'x+b'y=0 \end{equation*}
    from which we get
    \begin{equation*} a'x=-b'y \, . \end{equation*}
    However, \(a'\) and \(b'\) are coprime, and both \(a'x\) and \(b'y\) are multiples of \(\lcm(a',b')\text{.}\) Therefore, \(-b'y=a'x=t\lcm(a',b')=ta'b'\) for some \(t\text{,}\) which implies that any solution \((x,y)\) satisfies
    \begin{equation*} (x,y)=(tb',-ta')=t(b',-a') \, . \end{equation*}
    This shows that every solution to
    \begin{equation*} ax+by=0 \end{equation*}
    is of the form
    \begin{equation*} t(b',-a') \end{equation*}
    for some integer \(t\text{,}\) which completes the proof.
    ðŸ”—

    Example 4.4.6.

    To find all possible solutions to \(3x+8y=1\text{,}\) we first find a particular solution, which can be done by using the Euclidean algorithm:
    \begin{gather*} 8=3\cdot 2 + 2\\ 3=2\cdot 1 + 1\\ 2 = 1 \cdot 2 + 0 \end{gather*}
    Working backwards, we find
    \begin{equation*} 1 = 3-2\cdot 1 = 3-(8-3\cdot 2)\cdot 1 = 3\cdot 3 - 8 \end{equation*}
    from which it follows that \((x_0,y_0)=(3,-1)\) is a particular solution.
    ðŸ”—
    We then note that all homogeneous solutions are of the form \(t(8,-3)\) for \(t\in \Z\text{.}\) Thus, the full solution set is
    \begin{equation*} \{(3,-1)+t(8,-3):t\in \Z\}\, . \end{equation*}

    There is one final observation to make about the classification of solutions to linear Diophantine equations. We see that the full solution set is of the form
    \begin{equation*} \text{particular }+\text{ homogeneous} \end{equation*}
    which you will also see in MA 322 (Matrix Algebra) when solving systems of linear equations, and in MA 214 (Calculus IV) when solving linear systems of differential equations. This type of solution set is a characteristic of linear equations across multiple fields of mathematics, so you should keep watch for when this type of solution set appears.
