    Section 5.7 Musical Scales and Keys

    We will now apply modular arithmetic to the study of musical scales and keys. First, we will need to introduce some notation for notes in an equally-tempered chromatic scale.
    🔗
    Recall that a note is given by a frequency \(f\text{,}\) and that two frequencies are defined to be equivalent if they differ by a power of \(2\text{.}\) Observe that this is an equivalence relation:
    • Reflexivity: \(f=f\) for all frequencies \(f\text{.}\)
      🔗
      🔗
    • Symmetry: If \(f=2^ng\text{,}\) then \(2^{-n}f=g\text{.}\)
      🔗
      🔗
    • Transitivity: If \(f=2^ng\) and \(g=2^mh\text{,}\) then \(f=2^{n+m}h\text{.}\)
      🔗
      🔗
    🔗

    Checkpoint 5.7.1.

    Discuss the above proof that this is an equivalence relation. Does it make sense? Are you surprised that this idea of equivalence that we already encountered satisfies the reflexivity, symmetry, and transitivity axioms?
    🔗
    🔗

    Definition 5.7.2.

    Given any frequency \(f\) Hz, we will call the equivalence class \([f]\) under the above equivalence relation the "frequency class" of \(f\text{.}\) For the frequency \(440\) Hz, we will refer to the frequency class \([440]\) by the letter "A".
    🔗
    🔗
    Based on the definition of frequency class, we can now define a chromatic scale.
    🔗

    Definition 5.7.3.

    The chromatic scale is the following set of frequency classes:
    \begin{equation*} \{[(\sqrt[12]{2})^i \cdot 440]: i=0,1,2,\ldots, 11\} \end{equation*}
    🔗
    🔗
    There are 12 frequency classes in an equally-tempered chromatic scale. However, while we define the standard equally-tempered scale based on the "A" at 440 Hz, the convention is to begin the chromatic scale with the note "C" at approximately 261.62 Hz. The following table gives the 12 chromatic scale notes, where each of the frequency classes is assigned both a note name and a "pitch class" which is a number that we will explain following the table.
    🔗
    We can see the pitch classes arise as the exponents of \(\sqrt[12]{2}\) when applied to the frequency for C to obtain the frequencies for each other note. For example, the pitch class number for E is \(4\text{,}\) and indeed \([(\sqrt[12]{2})^4 \cdot 261.62\ldots]=[329.62\ldots]\text{.}\) Similarly, the pitch class number for G\(\sharp\) is \(8\text{,}\) and indeed \([(\sqrt[12]{2})^8 \cdot 261.62\ldots]=[415.30\ldots]\text{.}\)
    🔗

    Checkpoint 5.7.5.

    Explain why the frequency class \([466.16\ldots]\) is the same note as the pitch class we would get for the pitch class \(22\text{.}\)
    🔗
    🔗
    Let’s listen to the chromatic scale, starting with C and ascending by one note at a time until we reach the next C, which is an octave above the first C. Audio for the scale can be found at https://en.wikipedia.org/wiki/Chromatic_scale.
    🔗
    Next, we will give a mathematical description of this presentation of the chromatic scale.
    🔗

    Definition 5.7.6.

    Given \(a\in \mathbb{Z}/12\mathbb{Z}\text{,}\) we define the transposition by \(a\) to be the function \(T_a:\mathbb{Z}/12\mathbb{Z}\to \mathbb{Z}/12\mathbb{Z}\) defined by \(T_a(x)=x+a\text{.}\)
    🔗
    🔗
    The chromatic scale can be thought of as the sequence of pitch classes that we obtain by applying the function \(T_1\) repeatedly to the pitch class \(0\text{,}\) which corresponds to the note C. For example, the pitch class for E is \(4\text{,}\) and indeed
    \begin{equation*} T_1^4(0)=T_1(T_1(T_1(T_1(0))))=4 \, . \end{equation*}
    This is possible because the elements of \(\mathbb{Z}/12\mathbb{Z}\) are generated by repeatedly adding \(1\text{.}\)
    🔗

    🔗
    As the checkpoint above shows, there are exactly three other ways to generate the notes in the chromatic scale.
    • We can repeatedly apply the function \(T_5\) to the pitch class \(0\text{.}\)
      🔗
      🔗
    • We can repeatedly apply the function \(T_7\) to the pitch class \(0\text{.}\)
      🔗
      🔗
    • We can repeatedly apply the function \(T_{11}\) to the pitch class \(0\text{.}\)
      🔗
      🔗
    This is because these are the only numbers in \(\mathbb{Z}/12\mathbb{Z}\) that are coprime to \(12\text{.}\) If we use \(T_a\) for some \(a\) that is not coprime to \(12\text{,}\) then we only generate a proper subset of the chromatic scale, namely those numbers divisible by \(\gcd(a,12)\text{.}\)
    🔗
    Because \([11]=[-1]\) in \(\mathbb{Z}/12\mathbb{Z}\text{,}\) the function \(T_{11}\) is the same as the function \(T_{-1}\text{,}\) which is the function that subtracts 1 from its input. Thus, \(T_{11}\) generates the descending chromatic scale, while \(T_1\) generates the ascending chromatic scale.
    🔗
    Note that going up by 7 pitch classes corresponds to going up by a perfect fifth, while going up by 5 pitch classes corresponds to going up by a perfect fourth. Further, going up by 7 pitch classes corresponds to going down by 5 pitch classes. This is because \(-5\equiv 7 \pmod{12}\text{,}\) which means
    \begin{equation*} [7]=[-5] \end{equation*}
    in \(\mathbb{Z}/12\mathbb{Z}\text{.}\)
    🔗
    The sequence of pitch classes generated by repeatedly applying \(T_7\) to the pitch class \(0\) is called the circle of fifths because it corresponds to going up by perfect fifths:

    🔗
    Each individual note in the chromatic scale corresponds to a "major key", and each major key corresponds to a diatonic scale, defined as follows.
    🔗

    Definition 5.7.10.

    The (major) diatonic scale in the key of C is the scale consisting of the notes C, D, E, F, G, A, and B. Equivalently, the diatonic scale in the key of C is the set of pitch classes:
    \begin{equation*} \{0, 2, 4, 5, 7, 9, 11\} \end{equation*}
    The diatonic scale in the key for pitch class \(k\) is the set of pitch classes obtained by applying \(T_k\) to each pitch class in the key of C.
    🔗
    🔗
    For example, the diatonic scale in the key of A corresponds to the pitch classes obtained by applying \(T_9\) to each pitch class in the key of C, which yields:
    \begin{equation*} \{9,11,1,2,4,6,8\} \end{equation*}
    corresponding to the notes A, B, C\(\sharp\text{,}\) D, E, F\(\sharp\text{,}\) and G\(\sharp\text{.}\)
    🔗

    🔗
    The above exercise leads to the following theorem.
    🔗

    Theorem 5.7.12.

    Applying \(T_7\) to the pitch classes in a diatonic scale with pitch classes
    \begin{equation*} \{k,k+2,k+4,k+5,k+7,k+9,k+11\} \end{equation*}
    changes only one of the pitch classes, specifically the fourth pitch class in the diatonic scale.
    🔗
    🔗

    Proof.

    Applying \(T_7\) to the pitch classes in the diatonic scale with pitch classes
    \begin{equation*} \{k,k+2,k+4,k+5,k+7,k+9,k+11\} \end{equation*}
    yields the pitch classes
    \begin{equation*} \{k+7, k+9, k+11, k, k+2, k+4, k+6\} \end{equation*}
    which is the same as the set of pitch classes
    \begin{equation*} \{k, k+2, k+4, k+6, k+7, k+9, k+11\} \end{equation*}
    because we are working in \(\mathbb{Z}/12\mathbb{Z}\text{.}\) Thus, applying \(T_7\) changes only the fourth pitch class from \(k+5\) to \(k+6\text{.}\)
    🔗
    🔗

    Checkpoint 5.7.13.

    Does this proof make sense? Why or why not?
    🔗
    🔗
    The above theorem explains why the circle of fifths is so important in music theory. Each time we apply \(T_7\) to the pitch classes in a diatonic scale, we only change one of the pitch classes, which means that the diatonic scales corresponding to adjacent notes in the circle of fifths are very similar to each other. When we change musical keys by applying a transposition function, it is called "modulation" in music theory.
    🔗
    Each major key also has an associated "major triad", which is a chord consisting of three notes in the major key.
    🔗

    Definition 5.7.14.

    If the major key has pitch classes \(\{k,k+2,k+4,k+5,k+7,k+9,k+11\}\text{,}\) then the major triad for that key consists of the notes with pitch classes
    \begin{equation*} \{k,k+4,k+7\} \, . \end{equation*}
    🔗
    🔗

    Checkpoint 5.7.15.

    Find the major triads for the keys of C (\(k=0)\text{,}\) G (\(k=7\)), and D (\(k=2\)).
    🔗
    🔗
    In general, a triad is a chord (collection of notes played simultaneously) consisting of three notes. Each major key has seven triads associated to it, one for each of the seven notes in the major key, and these triads are called the "diatonic triads" for that key. The way to obtain a triad is to start at a note in the diatonic scale and then take the second and fourth notes in the diatonic scale after that note (where we wrap around to the beginning of the diatonic scale if necessary).
    🔗
    For example, the diatonic triads for the key of C are as follows. In this table, each column corresponds to a note in the key of C, the second row gives the pitch classes for the diatonic triad corresponding to that note, the third row gives the names of the notes in the triads, and the fourth row gives the Roman numeral for that chord.
    🔗
    The diatonic triads for the other major keys can be obtained by applying the appropriate transposition function to the pitch classes in the diatonic triads for the key of C. For example, the ii chord in the key of D corresponds to the triad with pitch classes obtained by applying \(T_7\) twice to the pitch classes in the ii chord \(\{2,5,9\}\) for the key of C, which yields the triad with pitch classes \(\{4,7,11\}\text{,}\) corresponding to the notes E, G, and B.
    🔗
    What we can see from this analysis is that the modular arithmetic structure of \(\mathbb{Z}/12\mathbb{Z}\) is fundamental to the structure of musical scales and keys, and that the circle of fifths arises specifically due to the fact that \(7\) is coprime to \(12\text{.}\) Further, the modular arithmetic structure of \(\mathbb{Z}/12\mathbb{Z}\) explains why the diatonic scales corresponding to adjacent keys in the circle of fifths are so similar to each other.
