    Section 6.3 Continued Fractions and Convergents

    In Problem P19 8.0.22, we saw that every rational number has an expansion as a finite continued fraction:
    \begin{equation*} \frac{a}{b} = q_0 + \cfrac{1}{ q_1 + \cfrac{1}{ q_2 + \cfrac{1}{ \ddots + \cfrac{1}{q_k} } } } \end{equation*}
    We will denote this continued fraction by \([q_0;q_1,q_2,\ldots,q_k]\text{.}\)
    🔗
    For this problem, we assumed that \(a\gt b\) were positive integers, thus we restricted ourselves to the case where a rational number is in the interval \((1,\infty)\text{.}\) However, we can find a continued fraction expansion for any rational number, because we can add or subtract an integer to a rational number and it only changes the value of \(q_0\) in the continued fraction expansion.
    🔗

    Checkpoint 6.3.1.

    Discuss the observation above with your group. Do you agree that if we have two rational numbers \(x\) and \(y\text{,}\) and if \(x=n+y\) for some integer \(n\text{,}\) then the continued fraction for \(x\) is obtained from the continued fraction for \(y\) by adding \(n\) to the \(q_0\) value for \(y\text{?}\)
    🔗
    🔗
    Next, let’s remind ourselves how this continued fraction expression was obtained on your homework. If \(a \gt b\) are positive integers, then we apply the Euclidean algorithm to get the following sequence:
    \begin{align*} a \amp = q_0b+r_1 \\ b \amp = q_1r_1 + r_2 \\ r_1 \amp = q_2r_2 + r_3 \\ r_2 \amp = q_3r_3 + r_4 \\ \vdots \amp = \vdots \\ r_{k-1} \amp = q_kr_k + 0 \end{align*}
    If you divide the first equation by \(b\) you obtain
    \begin{equation*} \frac{a}{b} = q_0 + \frac{r_1}{b} = q_0 + \frac{1}{\frac{b}{r_1}}\, . \end{equation*}
    If you divide the second equation by \(r_1\text{,}\) you obtain
    \begin{equation*} \frac{b}{r_1}=q_1+\frac{r_2}{r_1} = q_1+\frac{1}{\frac{r_1}{r_2}} \, , \end{equation*}
    which can be substituted into the first equation to obtain
    \begin{equation*} \frac{a}{b} = q_0 + \frac{r_1}{b} = q_0 + \frac{1}{q_1+\frac{1}{\frac{r_1}{r_2}}} \, . \end{equation*}
    We can continue in this way until we reach the last equation in the Euclidean algorithm, and thus we obtain a continued fraction representation for \(a/b\text{.}\) Thus, given any rational number, we can write it as a continued fraction where \(q_0\in \mathbb{Z}\) and each \(q_i\in \mathbb{Z}_{\geq 1}\text{.}\)
    🔗

    Also observe that if we consider any \(q_0\in \mathbb{Z}\) and let \(q_i\in \mathbb{Z}_{\geq 1}\) for \(i=1,2,\ldots,k\text{,}\) then the value
    \begin{equation*} q_0 + \cfrac{1}{ q_1 + \cfrac{1}{ q_2 + \cfrac{1}{ \ddots + \cfrac{1}{q_k} } } } \end{equation*}
    is a rational number. This is obtained by recursively (i.e., inductively) finding a common denominator at each level. For example, if we consider the number
    \begin{equation*} q_0 + \cfrac{1}{ q_1 + \cfrac{1}{ q_2 + \cfrac{1}{q_3} } } \, , \end{equation*}
    then we have
    \begin{equation*} q_0 + \cfrac{1}{ q_1 + \cfrac{1}{ \frac{q_2q_3+1}{q_3} } } \end{equation*}
    from which we obtain
    \begin{equation*} q_0 + \cfrac{1}{ q_1 + \cfrac{q_3}{q_2q_3+1} } \end{equation*}
    from which we obtain
    \begin{equation*} q_0 + \cfrac{1}{\frac{q_1(q_2q_3+1)+q_3}{q_2q_3+1}} \end{equation*}
    from which we obtain
    \begin{equation*} q_0 +\frac{q_2q_3+1}{q_1(q_2q_3+1)+q_3} \end{equation*}
    from which we obtain
    \begin{equation*} \frac{q_0(q_1(q_2q_3+1)+q_3)+q_2q_3+1}{q_1(q_2q_3+1)+q_3} \end{equation*}
    which is a rational number since the denominator only involves products and sums of positive integers, hence is not equal to zero. This is an embodied-symbolic proof of this fact; a fully formal proof would provide a proof by induction.
    🔗

    Checkpoint 6.3.4.

    🔗
    Summarizing the two arguments above, we have proved the following.
    🔗

    Theorem 6.3.5.

    A number is rational if and only if it has a finite continued fraction expansion.
    🔗
    🔗

    Checkpoint 6.3.6.

    Discuss the theorem above. Does the proof preceding it make sense?
    🔗
    🔗
    You obtained in problem Problem P23 8.0.26 an infinite continued fraction representation for \(\sqrt{2}\text{.}\) It turns out that every real number has a continued fraction expansion. We will denote such an expansion using the following notation.
    🔗

    Definition 6.3.7.

    If \(q_0\in \mathbb{Z}\) and \(q_i\in \mathbb{Z}_{\geq 1}\) for all \(i=1,2,3,\ldots\text{,}\) we denote by
    \begin{equation*} [q_0;q_1,q_2,q_3,\ldots] \end{equation*}
    the infinite continued fraction
    \begin{equation*} q_0 + \cfrac{1}{ q_1 + \cfrac{1}{ q_2 + \cfrac{1}{ \ddots} } } \end{equation*}
    which is defined to be the value
    \begin{equation*} \lim_{n\to \infty}[q_0;q_1,q_2,\ldots,q_n] \, . \end{equation*}
    🔗
    🔗
    The following theorem can be proved using a combination of technical details about continued fractions and techniques from real analysis to study sequence convergence (which you can learn more about in MA 471 or MA 575). Thus, the full proof is beyond the scope of this course.
    🔗

    Theorem 6.3.8.

    Every irrational number admits a unique representation as an infinite continued fraction, and every infinite continued fraction converges to a unique irrational number.
    🔗
    🔗
    Let’s look at the continued fraction expansion for \(\sqrt{3}\approx 1.73205\text{.}\) Note that this means we have
    \begin{align*} \sqrt{3} \amp \approx 1+0.73205 \\ \amp = 1+\frac{1}{1.36603} \\ \amp = 1+\frac{1}{1+\frac{1}{2.73205}}\\ \amp = 1+\frac{1}{1+\frac{1}{2+\frac{1}{1.3660}}} \end{align*}
    and thus we have that the beginning of the resulting infinite continued fraction is \(\sqrt{3}=[1;1,2,1,\ldots]\text{.}\)
    🔗

    Checkpoint 6.3.9.

    If we have an irrational number given by \(s=[q_0;q_1,q_2,q_3,\ldots]\text{,}\) then we define the \(m\)-th convergent of \(s\) to be the rational number
    \begin{equation*} \frac{b_m}{c_m}:= [q_0;q_1,q_2,q_3,\ldots,q_m] \, . \end{equation*}

    The key to obtaining best approximations of an irrational number \(s\) is to look at the convergents of \(s\text{,}\) as the following theorem shows.
    🔗

    Theorem 6.3.12.

    Let \(s\) be an irrational number. A rational number is a best approximation of the second kind for \(s\) if and only if it is a convergent for \(s\text{.}\)

    🔗
    The proof of this theorem is beyond the scope of our course, and can be found in the book Continued Fractions by A. Ya. Khinchin, available at this link. However, this theorem motivates a further study of convergents, leading to the following theorem that gives a recursive method for computing convergents for an infinite continued fraction.
    🔗

    Theorem 6.3.14.

    Given an infinite continued fraction \([q_0;q_1,q_2,q_3,\ldots]\text{,}\) define numbers \(b_m\) and \(c_m\) inductively as follows:
    \begin{align*} b_0 \amp =q_0\\ c_0 \amp = 1\\ b_1 \amp = q_1q_0+1 \\ c_1 \amp = q_1 \\ b_m \amp = q_mb_{m-1} + b_{m-2} \text{ for } m\geq 2 \\ c_m \amp = q_mc_{m-1} + c_{m-2} \text{ for } m\geq 2 \end{align*}
    Then we have
    \begin{equation*} \frac{b_m}{c_m} = [q_0;q_1,q_2,\ldots,q_m] \end{equation*}
    🔗
    🔗
    Let’s look at an example before we prove this theorem. It is known that \(\frac{1+\sqrt{5}}{2}=[1;1,1,1,1,\ldots]\text{.}\) Then we have
    \begin{equation*} b_0,b_1,b_2,b_3,\ldots = 1,2,3,5,8,13,\ldots \end{equation*}
    since \(b_m=b_{m-1}+b_{m-2}\) and
    \begin{equation*} c_0,c_1,c_2,c_3,\ldots = 1, 1,2,3,5,8,13,\ldots \end{equation*}
    since \(c_m=c_{m-1}+c_{m-2}\text{.}\) Thus, the convergents of \(\frac{1+\sqrt{5}}{2}\) are of the form \(\frac{F_{m}}{F_{m-1}}\) where \(F_m\) is the \(m\)-th Fibonacci number.
    🔗

    Checkpoint 6.3.15.

    Discuss the example above with your group. Does it make sense? Why or why not?
    🔗
    🔗

    Checkpoint 6.3.16.

    Use the above recursion to compute the first six convergents of \(\pi=[3;7,15,1,292,1,1,1,\ldots]\text{.}\)
    🔗
    🔗

    Proof.

    For the base cases of \(m=0,1\text{,}\) we have \(b_0/c_0=q_0\) and \(b_1/c_1=q_0+\frac{1}{q_1}\text{,}\) from which the formulas in the theorem follow. When \(m=2\text{,}\) we see that
    \begin{equation*} \frac{b_2}{c_2}=\frac{q_0(q_1q_2+1)+q_2}{q_1q_2+1} \, . \end{equation*}
    This establishes the base cases for \(m\leq 2\text{.}\)
    🔗
    We next proceed by induction. Assume that the statement is true up to \(m\) for all continued fractions, and we will show it for \(m+1\text{.}\) First, observe that if
    \begin{equation*} \frac{b}{c}=[q_0;q_1,\ldots,q_n] \end{equation*}
    and
    \begin{equation*} \frac{b'}{c'}=[q_1;q_2,\ldots,q_n] \, , \end{equation*}
    then we have
    \begin{equation*} \frac{b}{c}=q_0+\frac{1}{\frac{b'}{c'}}=\frac{b'q_0+c'}{b'} \, , \end{equation*}
    and thus we have what we will call the shifting relation:
    \begin{align*} b \amp = b'q_0+c' \\ c \amp = b' \end{align*}
    With this in mind, if we have the equations
    \begin{align*} \frac{b_{m+1}}{c_{m+1}} \amp =[q_0;q_1,\ldots,q_{m+1}]\\ \frac{\beta_j}{\gamma_j} \amp = [q_1;q_2,\ldots,q_{j+1}] \end{align*}
    where the second index is \(j\) since the sequence starts with \(q_1\text{,}\) then:
    \begin{align*} b_{m+1} \amp = q_0\beta_m+\gamma_m\\ c_{m+1} \amp = \beta_m \end{align*}
    By induction, we have:
    \begin{align*} \beta_m \amp = q_{m+1}\beta_{m-1}+\beta_{m-2} \\ \gamma_m \amp = q_{m+1}\gamma_{m-1}+\gamma_{m-2} \end{align*}
    Now, plugging these expressions into our shifting relation for \(m+1\text{,}\) we obtain:
    \begin{align*} b_{m+1} \amp = q_{0}(q_{m+1}\beta_{m-1}+\beta_{m-2})+ q_{m+1}\gamma_{m-1}+\gamma_{m-2}\\ \amp = q_{m+1}(q_{0}\beta_{m-1}+\gamma_{m-1}) + q_0\beta_{m-2}+\gamma_{m-2} \\ \amp = q_{m+1} b_m+b_{m-1} \end{align*}
    and
    \begin{align*} c_{m+1} \amp = q_{m+1}\beta_{m-1}+\beta_{m-2}\\ \amp = q_{m+1}c_{m}+c_{m-1} \end{align*}
    which agrees with the sequences we defined in the theorem statement, completing the proof.
